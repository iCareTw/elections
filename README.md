# 選舉候選人資料庫

整合歷屆選舉候選人資訊, 產出結構化 YAML 供其他專案使用.

## 資料來源

原始資料取自 **中央選舉委員會 (中選會)** 公開資料:

- 網站: https://www.cec.gov.tw/
- 資料集: 歷屆選舉候選人名單 (.xlsx)

涵蓋選別:

| 選別       | 說明                         |
|------------|------------------------------|
| 總統副總統 | 歷屆正副總統候選人           |
| 縣市首長   | 直轄市市長、縣 (市) 長候選人 |
| 立法委員   | 區域、不分區立法委員候選人   |

## 產出文件

| 檔案                  | 說明                               |
|-----------------------|------------------------------------|
| `candidates.yaml`     | 候選人主 mapping (姓名、id、參選紀錄) |
| `election_types.yaml` | 合法選別 enum 清單                 |

## 免責聲明

- 本資料庫以中選會公開資料為基礎整理而成, 僅供研究、學術及非商業用途參考.
- 資料整合過程可能存在疏漏或錯誤, **不保證資料之完整性與正確性**.
- 本資料庫**仍在持續整合中**, 部分選別或屆次資料尚未納入.
- 如需用於正式、法律或選務相關用途, 請以中選會官方公告為準.
- 本專案與中央選舉委員會無任何官方關聯.
